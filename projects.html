OpenVolumetricMesh (OVM)--a generic and efficient data structure for tetrahedral and hexahedral mesh

In this project, we implement the half-face data structure for the static representation and the dynamic handling of tetrahedral and hexahedral mesh. The objects of the design of the data structure aim at maximum flexibility and high performance. More details can be found in the OVM project page.
